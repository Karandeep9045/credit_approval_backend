Welcome to the Credit Approval System API. Below you'll find the available API endpoints and their descriptions.

> API Endpoints & Sample Requests

Base URL: http://127.0.0.1:8000
  • POST /api/register — Register a new customer
    Sample Request: 
    {
  "first_name": "John",
  "last_name": "Doe",
  "phone_number": "9876543210",
  "monthly_income": 75000,
  "age": 28
}
  • POST /api/check-eligibility — Check loan eligibility
    Sample Request: 
    {
  "customer_id": 3,
  "loan_amount": 200000,
  "interest_rate": 10.0,
  "tenure": 24
}
  • POST /api/create-loan — Create a new loan
    Sample Request: 
    {
  "customer_id": 3,
  "loan_amount": 200000,
  "interest_rate": 10.0,
  "tenure": 24
}
  • GET /api/view-loan/<loan_id> — View details of a specific loan
    Sample: GET /api/view-loan/3050
  • GET /api/view-loans/<customer_id> — View all loans for a specific customer
    Sample: GET /api/view-loans/3

> Notes

  • Ensure Celery & Redis are running for background ingestion.
  • Use Postman or Swagger to test APIs.
  • All responses follow JSON format with proper HTTP status codes.
